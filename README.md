# 🎓 R3F Study Project

Projeto de estudo utilizando **React Three Fiber (R3F)** para aprender conceitos de renderização 3D no React.

👉 **Acesse o projeto online:**  
🔗 *https://luscap3.github.io/R3F-Study/*

---

## 🧠 Objetivo

Explorar e praticar:

- Renderização 3D com React
- Uso do Three.js de forma declarativa
- Manipulação de câmera e objetos
- Animações com `useFrame`

---

## 🚀 Tecnologias

- React
- React Three Fiber
- @react-three/drei
- Three.js
- Vite
